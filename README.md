The project includes:

* Installation and organization of multiple Dell servers in a small server rack
* Cisco switch and router connectivity
* Patch-panel installation and port organization
* Custom Ethernet cables created in different lengths
* RJ45 measuring, cutting, stripping, termination, crimping, and testing
* Cable labelling and physical-layer troubleshooting
* Proxmox installation and virtual machine deployment
* CPU, memory, storage, and virtual network allocation
* Multiple Linux physical and virtual system installations
* NAS configuration and shared storage access
* RAID capacity, redundancy, performance, and failure-protection comparisons
* Server, network, virtualization, and storage troubleshooting
* Architecture diagrams, technical documentation, and privacy-conscious evidence

This is an independent home-lab and portfolio project created for continuous technical development. It does not represent production enterprise administration or paid infrastructure experience.

* MAC addresses
* Serial numbers
* Dell service tags
* Asset tags
* Wi-Fi network names
* Passwords
* Usernames
* Email addresses
* API keys
* Storage credentials
* Private keys
* Exact physical location
* Full router, firewall, switch, NAS, or hypervisor configuration files

Only sanitized, authorized, and public-safe technical evidence is included.
